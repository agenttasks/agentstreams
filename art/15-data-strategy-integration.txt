    Data Flow:
    ──────────
    1. PluginLoader reads vendors/ → SKILL_CATALOG populated
    2. SKILL_CATALOG → seeds ontology/migrations/001-*.sql
    3. skills.sh crawl → taxonomy/skills-sh.md (install counts)
    4. Claude Code docs → taxonomy/code-claude-com.md (API patterns)
    5. Managed Agents docs → taxonomy/platform-claude-com.md
    6. All feed → Neon Postgres via ontology/schema.sql tables
    7. pg_graphql exposes everything via GraphQL API
