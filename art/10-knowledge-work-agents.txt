    Server-Managed Settings Integration
    ─────────────────────────────────────
    Plugins deploy org-wide via server-managed settings
    (code.claude.com managed-agents-2026-04-01 beta):

    {
      "permissions": {
        "deny": ["Bash(curl *)", "Read(./.env)", "Read(./.env.*)"]
      },
      "allowManagedPermissionRulesOnly": true
    }

    Installation
    ────────────
    # Cowork
    Install from claude.com/plugins/

    # Claude Code
    claude plugin marketplace add anthropics/knowledge-work-plugins
    claude plugin install sales@knowledge-work-plugins
